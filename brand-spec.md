# KENAZ — Brand Spec (Level 2 preview page)

Extracted from `kenaz-brand-reference.md` (swept from kenazportal.com). Charcoal + gold
on white/off-white, Raleway 300, spa-luxe minimal. Gold is the single accent, used sparingly.

## Color tokens (OKLch)

```css
:root {
  --bg:      oklch(98.5% 0 0);      /* #fafafa off-white page */
  --surface: oklch(100% 0 0);       /* #ffffff cards / sections */
  --fg:      oklch(27% 0 0);        /* #212121 charcoal text */
  --muted:   oklch(54% 0 0);        /* #737373 secondary text */
  --border:  oklch(89% 0 0);        /* #d9d9d9 hairlines */
  --accent:  oklch(76% 0.12 86);    /* #d4af37 signature gold */
  --ink:     oklch(18% 0 0);        /* #1b1b1b dark sections */
}
```

## Type

- Display + body: **Raleway 300 (Light)**. Fallback `Raleway, "Nunito Sans", "Open Sans", sans-serif`.
- Weights in use: 300 (default), 400, 600 (rare). Keep it thin and airy.
- Uppercase eyebrows: tracked `0.28–0.34em`, gold or muted.

## Layout posture

1. Hairlines, not boxes — borders + whitespace do the structural work; almost no shadow.
2. Gold appears only as: the chevron mark, hairline accents/dividers, and the CTA.
3. Generous vertical rhythm; full-bleed natural-light photography with soft charcoal overlays.
4. Selective ALL-CAPS on charged words (BREATH, DEEPER, ALIVENESS) inside running copy.
5. The KENAZ arc — SEE → FEEL → EMBRACE — is the connective tissue of the journey.
